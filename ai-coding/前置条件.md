SpeakCoach — 前置资源收集清单

开发启动前，请两人共同完成以下申请，填入 .env。  
全程可用免费额度，无需付费。

---

1. Azure Speech Services（必须，所有语音能力）

申请步骤
1. 登录 Azure Portal（无账号则免费注册）
2. 搜索 "Speech Services" → 点击 Create
3. 配置：
   - Resource group：新建或选已有
   - Region：选 `East Asia` 或 `East US`（影响延迟）
   - Pricing tier：选 `Free F0`（5 小时/月 STT，50 万字符/月 TTS，足够 demo）
4. 创建完成后进入资源 → Keys and Endpoint

需要收集
AZURE_SPEECH_KEY=<KEY1 或 KEY2，任选其一>
AZURE_SPEECH_REGION=<e.g. eastasia 或 eastus>

包含能力（一个 key 全搞定）
功能
API
Demo 用量估计
STT 语音转文字
Speech-to-Text REST
~2小时/天
发音评测
Pronunciation Assessment (SDK)
同 STT 配额
TTS 文字转语音
Text-to-Speech
~10万字符/天

Python 安装
pip install azure-cognitiveservices-speech

注意
- Pronunciation Assessment 要求音频为 PCM WAV 16kHz 16-bit 单声道
- 浏览器录制的 webm/opus 需先用 pydub 转换（代码已在 A 模块提供）
- Free F0 有并发限制（1 并发），Demo 单用户无问题

2. LLM API（必须，对话生成 + 语法纠错）

选项 A：DeepSeek（推荐）
1. 注册 platform.deepseek.com
2. 进入 API Keys → Create new key
3. 新用户赠送 10 元额度（足够 demo 全程）
LLM_API_KEY=<deepseek api key>
LLM_BASE_URL=https://api.deepseek.com/v1
LLM_MODEL=deepseek-chat

选项 B：阿里云通义千问
1. 注册 dashscope.aliyuncs.com
2. API Keys → 创建
3. 新用户赠送 token 额度
LLM_API_KEY=<dashscope api key>
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL=qwen-turbo

5. 验证清单（环境就绪检查）

[] AZURE_SPEECH_KEY 填入，Azure Portal 状态为 Active
[] 执行 Python 测试脚本可完成一次 STT 识别并返回文字
[] LLM_API_KEY 填入，curl 调用 /v1/chat/completions 有正常回包
[] ffmpeg -version 命令可执行
[] python --version ≥ 3.11，node --version ≥ 18
Azure STT 快速验证脚本
import azure.cognitiveservices.speech as speechsdk

def test_azure():
    cfg = speechsdk.SpeechConfig(
        subscription="YOUR_KEY",
        region="YOUR_REGION"
    )
    recognizer = speechsdk.SpeechRecognizer(speech_config=cfg)
    print("Speak something...")
    result = recognizer.recognize_once()
    print(f"Recognized: {result.text}")

test_azure()

DeepSeek 快速验证脚本
from openai import OpenAI

client = OpenAI(api_key="YOUR_KEY", base_url="https://api.deepseek.com/v1")
resp = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "Say hello in English"}],
)
print(resp.choices[0].message.content)
