Innovation 1：AI 角色个性化 + 场景难度闯关

功能描述
- 每个场景配置多个个性化 AI 角色（如面试场景有"严厉技术官"和"友好HR"两种角色）
- 难度分 3 级（1=基础词汇、2=连贯表达、3=复杂句式），通过后解锁下一级
- 每轮结束展示是否达到本级"闯关标准"（发音均分 ≥ 阈值）
挂载点
位置
改动
backend/app/core/scenes.py
每个 scene 增加多 persona + difficulty_config 字段
backend/app/modules/conversation/llm_client.py
system prompt 读取 persona 的 style_instructions
frontend/src/modules/conversation/SceneSelector.vue
增加角色卡片选择 + 难度星级选择 UI
frontend/src/modules/coach/SessionSummaryPanel.vue
增加"闯关结果"模块

关键数据结构扩展
# scenes.py 新增字段
"personas": {
    "strict_interviewer": {
        "name": "Alex",
        "avatar": "👔",
        "style_instructions": "You are Alex, strict and demanding. Probe for weaknesses. Push back on vague answers.",
        "difficulty_thresholds": {1: 60, 2: 72, 3: 85}  # pron_score 闯关阈值
    }
}

估时
- Backend 场景配置扩展：0.5h
- 前端 SceneSelector UI 升级：1.5h
- 闯关判断逻辑 + 总结展示：1h
- 合计：~3h

Innovation 2：实时纠错气泡（对话中即时提示）

功能描述
- 每轮 AI 回复播放时，右下角弹出浮动气泡
- 气泡显示最重要的一条语法错误 + 更好的说法
- 点击气泡可展开完整纠错列表
- 3秒后自动收起（不打断对话节奏）
挂载点
位置
改动
frontend/src/modules/coach/CorrectionPanel.vue
改为 fixed 定位气泡，加 CSS 动画
frontend/src/core/store.ts
增加 latestCorrectionBubble state
无需改后端
correction WS 事件格式不变，只改前端展示

关键实现
// CorrectionBubble.vue (新文件，Dev B 模块内)
// 监听 store.currentCorrections 变化
// 显示 corrections[0] 作为气泡主内容
// 用 Vue transition + Tailwind 动画效果
watch(() => store.currentCorrections, (corrections) => {
  if (corrections.length > 0) {
    showBubble.value = true
    setTimeout(() => { showBubble.value = false }, 3000)
  }
})

估时
- 气泡 Vue 组件 + 动画：1.5h
- Store 配合改动：0.5h
- 合计：~2h

Innovation 4：错题本 + 个性化复习

功能描述
- 自动收集历次高频错误（相同语法问题出现 ≥ 2 次则进入错题本）
- 错题本页面展示错误模式 + 正确示例
- 点击"复习"按钮，AI 自动出 3 个含该错误点的造句练习
挂载点
位置
改动
frontend/src/modules/coach/
新增 ErrorBook.vue + useErrorBook.ts
frontend/src/core/store.ts
增加 errorBook state（localStorage 持久化）
backend/app/modules/coach/summary_engine.py
生成 summary 时附带 frequent_errors 字段

关键数据结构
interface ErrorEntry {
  pattern: string        // e.g. "混淆 interested/interesting"
  example: string        // "I am interest in..." → "I am interested in..."
  count: number          // 出现次数
  last_seen: string      // ISO date
  review_sentences: string[]  // 复习句（LLM 生成）
}

估时
- localStorage 错题本逻辑：1h
- ErrorBook.vue 展示页：1.5h
- 后端 frequent_errors 生成：1h
- 复习句 LLM 生成接口：1h
- 合计：~4.5h
